HALLO, TERIMAKASIH SUDAH MAU MENGKLIK FILE INI, DISINI ADA BEBERAPA HAL YANG PERLU SAYA SAMPAIKAN.

MENGENAI :
1. FILE INI AKAN MENJADI PRIVATE SELAMA MASIH DALAM MASA CEK OLEH TIM DARI DICODING 
(KARENA MASIH ADA KEMUNGKINAN KESALAHAN PENULISAN KODE ATAU YANG LAINNYA YANG MUNGKIN TIDAK SAYA SENGAJA)

2. KETIKA SUDAH DI ACC OLEH TIM DARI DICODING DAN LULUS, FILE INI AKAN MENJADI PUBLIC
(FILE AKAN SAYA UPLOAD DI GITHUB DAN HOSTING MENGGUNAKAN NETLIFY)

RULES :
DENGAN TIDAK MENGURANGI RASA HORMAT SAYA KEPADA SELURUH TIM DARI DICODING, SAYA INGIN MEMPUBLISH SEMUA CODE INI UNTUK PEMBELAJARAN, NAMUN JANGAN KHAWATIR
CODE JAVASCRIPT SEBAGIAN AKAN SAYA ENKRIPSI UNTUK MENCEGAH PESERTA LAIN YANG MENGIKUTI KELAS YG SAMA DENGAN SAYA MEMANFAATKAN SOURCE CODE YG ADA DI GITHUB SAYA.

1. KODE AKAN DI ENKRIPSI
2. KODE YG DI ENKRIPSI AKAN DIACAK SEHINGGA YG BELUM BENAR-BENAR PAHAM AKAN KEBINGUNGAN MEMBACA KODE SAYA
3. KODE AKAN SAYA PROTECT DI SATU FILE (DI BAGIAN LOCAL DATA STORAGE)

SELURUH KODE YG ADA DISINI HASIL SAYA BELAJAR DARI DICODING, TERIMA KASIH BANYAK UNTUK ITU, DAN JUGA TERIMA KASIH TELAH MAU MEMBACA INI :)
